# NumberGuessingGame

Description
A Simple Command-Line Number Guessing Game using Java
Allows user to choose difficulty and guess a number 
randomly generated by the computer. User can choose
to play again after each win/loss.

Features:
> User input/console output
> Tracking statistics
> Difficulty level choice

How to Run:
javac Main.java
java Main

What I Learned:
> Exception handling
> Helper methods
> User input
> Conditional statements
